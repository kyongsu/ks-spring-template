## Git Commit Message Conventions
- feat <- 새로운기능
- fix <- 수정
- style <- 리팩토링
- chore <- 관리적(빌드등..)
```
example

   1. fix: OO 오류로 OO 기능수정
   2. fix(기능명): OO 오류로 기능 수정
      상세내용
      ...
   3. fix: OOO 오류 수정
      close: #100
```
